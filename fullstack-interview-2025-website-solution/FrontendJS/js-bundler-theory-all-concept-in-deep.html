1. What do you know about JavaScript?

                1. Core Concepts

                 • Data types (primitive & reference)
                 • Variables (var, let, const) and scope
                 • Hoisting
                 • Closures and lexical scope
                 • Prototype and inheritance
                 • The this keyword and its behavior in different contexts
                 • ES6+ features: destructuring, template literals, arrow functions, modules, spread/rest, etc.

            2. Asynchronous JavaScript

                 • Callbacks, Promises, async/await
                 • Event loop and microtasks/macrotasks
                 • APIs like fetch(), setTimeout(), and setInterval()
                 • Error handling in async functions

            3. DOM & Browser APIs

                 • Document Object Model manipulation
                 • Event bubbling and capturing
                 • Local storage, session storage, and cookies
                 • Working with Fetch API and AJAX

            4. Advanced Topics

                 • Closures, currying, memoization
                 • Higher-order functions (map, filter, reduce)
                 • Debouncing and throttling
                 • Module bundlers (Webpack, Vite)
                 • Understanding of runtime environments (Browser vs Node.js)

            5. JavaScript Ecosystem

               • Node.js and npm
                 • Frameworks/libraries: React, Angular, Vue
                 • TypeScript (optional but good to mention)
                 • Testing (Jest, Mocha, etc.)

            ⚡ Bonus Tip for Interviews:

            “I also stay updated with new ECMAScript features and best practices to write clean, efficient, and maintainable JavaScript code.”

  2. ES6 ?

                    ES6, also called ECMAScript 2015, is a major update to JavaScript that introduced features to make code cleaner and more powerful. 
                It added let and const for better variable scope, arrow functions for shorter syntax, template literals for string formatting, destructuring, 
                classes, modules, and promises for handling asynchronous code.These features made JavaScript more modern and developer-friendly
