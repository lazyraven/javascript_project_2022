1. Core Java (Foundation)

Focus on fundamentals and small CLI practice programs to build logic and confidence.

  • Data types, loops, operators
  • OOP: Encapsulation, Inheritance, Polymorphism, Abstraction
  • Exception handling
  • Collections Framework — List, Set, Map, Queue
  • Generics
  • Multithreading & Concurrency
  • File I/O (FileReader, BufferedReader, FileWriter)
  • Java 8+ features: Stream API, Lambdas, Optional, Functional Interfaces

Practice: reverse strings, find duplicates, sorting, map counting — write small CLI programs.

2. Advanced Java

  • JDBC (database connectivity)
  • Servlets and JSP (understand web flow)
  • HTTP Request/Response cycle, Filters, Listeners, Sessions

3. Frameworks — Spring & Spring Boot (Most Important)

  • Spring Core — IoC, Beans, Dependency Injection
  • Spring Boot — Auto-configuration, Starters, application.properties
  • Spring MVC — Controllers, RestControllers, @RequestMapping
  • Spring Data JPA — Repository pattern, CRUD
  • Spring Security — JWT, Authentication & Authorization

4. Database

  • SQL basics: Joins, Indexes, Stored Procedures
  • ORM: Hibernate or JPA
  • Practice CRUD with MySQL / PostgreSQL

5. REST API Development

  • Build RESTful APIs with Spring Boot
  • Use @GetMapping, @PostMapping, @PutMapping, @DeleteMapping
  • Understand JSON, DTOs, ResponseEntity
  • Global exception handling with @ControllerAdvice

6. Tools & DevOps Basics

  • Maven / Gradle
  • Postman for API testing
  • Git / GitHub
  • Docker basics (optional but recommended)

7. Project Layering

Follow a clean architecture for maintainability:

  • controller → service → repository → model → dto

Java Coding Practice (Interview Example)

Problem: Find the first non-repeated character in a string.

// FirstNonRepeatedChar.java
import java.util.LinkedHashMap;
import java.util.Map;
import java.util.Scanner;

public class FirstNonRepeatedChar {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter a string: ");
        String input = sc.nextLine();

        char result = findFirstNonRepeated(input);
        if (result != '\0') {
            System.out.println("First non-repeated character: " + result);
        } else {
            System.out.println("No unique character found.");
        }
    }

    public static char findFirstNonRepeated(String str) {
        Map map = new LinkedHashMap<>();
        for (char ch : str.toCharArray()) {
            map.put(ch, map.getOrDefault(ch, 0) + 1);
        }
        for (Map.Entry entry : map.entrySet()) {
            if (entry.getValue() == 1) {
                return entry.getKey();
            }
        }
        return '\0';
    }
}

Mini Project Example — Employee Management REST API

Stack: Spring Boot, MySQL, JPA

// Employee.java
@Entity
public class Employee {
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    private String name;
    private String role;
    private Double salary;

    // Getters and Setters
}

// EmployeeRepository.java
public interface EmployeeRepository extends JpaRepository { }

// EmployeeService.java
@Service
public class EmployeeService {
    @Autowired
    private EmployeeRepository repo;

    public List getAllEmployees() { return repo.findAll(); }
    public Employee addEmployee(Employee emp) { return repo.save(emp); }
    public Employee updateEmployee(Long id, Employee emp) { emp.setId(id); return repo.save(emp); }
    public void deleteEmployee(Long id) { repo.deleteById(id); }
}

// EmployeeController.java
@RestController
@RequestMapping("/api/employees")
public class EmployeeController {
    @Autowired
    private EmployeeService service;

    @GetMapping
    public List getAll() { return service.getAllEmployees(); }

    @PostMapping
    public Employee add(@RequestBody Employee emp) { return service.addEmployee(emp); }

    @PutMapping("/{id}")
    public Employee update(@PathVariable Long id, @RequestBody Employee emp) { return service.updateEmployee(id, emp); }

    @DeleteMapping("/{id}")
    public String delete(@PathVariable Long id) { service.deleteEmployee(id); return "Employee deleted with ID: " + id; }
}

Quick Tips

  • Practice small programs daily — consistency beats intensity.
  • Build one full CRUD project end-to-end (local DB & Postman tests).
  • Write unit tests (JUnit) and add simple integration tests.
  • Learn to read Spring Boot logs and common debugging patterns.

Commands & Tools

  • mvn spring-boot:run — run app
  • docker build -t myapp . — build image
  • Use Postman or curl for endpoint testing
  • Commit often with clear messages
