20-Day Interview Prep Calendar

Day 1: JS Basics

Key Focus Points: var/let/const, data types, operators, loops, conditionals

Day 2: Functions & Scope

Key Focus Points: Regular/arrow/IIFE, callbacks, closures, this

Day 3: Objects & Arrays

Key Focus Points: Object/array methods, destructuring, spread/rest

Day 4: ES6+ Features

Key Focus Points: Template literals, default params, modules, import/export

Day 5: Async JS

Key Focus Points: Promises, async/await, fetch, try/catch, event loop

Day 6: React Basics

Key Focus Points: JSX, Virtual DOM, components (functional/class), props & state

Day 7: React Hooks

Key Focus Points: useState, useEffect, conditional rendering, lists

Day 8: Forms & Events

Key Focus Points: Controlled/uncontrolled inputs, event handling, lifting state

Day 9: Advanced React

Key Focus Points: Context API, prop drilling, lifecycle methods, error boundaries

Day 10: React Performance

Key Focus Points: memo, lazy loading, Suspense, custom hooks, testing basics

Day 11: Node.js Basics

Key Focus Points: Architecture, event loop, modules (require/module.exports)

Day 12: Node Core Modules

Key Focus Points: fs, path, os, http

Day 13: Express.js

Key Focus Points: Routing, middleware, REST API CRUD operations

Day 14: Node Advanced

Key Focus Points: Error handling, authentication (JWT/cookies/sessions), env variables

Day 15: Async Node

Key Focus Points: Callbacks, promises, async/await, streams, buffers, deployment basics

Day 16: MongoDB Basics

Key Focus Points: Collections, documents, CRUD, data types, indexes

Day 17: Mongoose

Key Focus Points: Schemas, models, validation, middleware

Day 18: Queries & Aggregation

Key Focus Points: Filtering, sorting, projection, aggregation pipeline basics

Day 19: Relationships & Transactions

Key Focus Points: Referencing vs embedding, transactions, sessions

Day 20: Full-Stack Integration

Key Focus Points: Node + Express + MongoDB, performance optimization, final recap

Extra Tips

  • Allocate 2–3 hours/day for focused study + practice coding.
  • After day 10, start building a mini React + Node + MongoDB project while revising topics.
  • Solve JS algorithm questions daily (arrays, strings, recursion).
  • Record important interview Q&A for quick revision before the interview.
