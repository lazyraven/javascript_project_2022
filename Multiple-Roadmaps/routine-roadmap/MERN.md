# 📝 MERN Blog Platform

A full-stack blogging platform where users can create, edit, and manage blogs with secure authentication and optimized APIs.

---

## 📌 Problem

Many beginner blog apps lack proper authentication and scalable backend design. This project solves that by implementing **secure auth + structured APIs**.

---

## 🎯 Features

* JWT Authentication
* Create/Edit/Delete blogs
* Rich text editor
* Pagination & search
* Role-based access

---

## 🧠 Key Learnings

* Implemented JWT auth with middleware
* Designed REST APIs with proper structure
* Optimized MongoDB queries using indexing

---

## 🏗️ System Design

* Stateless backend using JWT
* MVC architecture
* Indexed fields for faster queries

---

## ⚙️ Tech Stack

* React, Node.js, Express, MongoDB

---

## 🚀 Future Scope

* Add comments & likes
* Add caching layer (Redis)
* Deploy with CI/CD

---

## 📊 Highlights

* Clean folder structure
* Scalable backend
* Real-world features
