     5️⃣ Tricky Interview Questions (Conceptual)
     ❓ Why indexOf() returns -1?
      Because indexes start from 0, and -1 clearly means not found.

      ❓ Why not use if (str.indexOf("a"))?
        // ❌ Wrong
        if (str.indexOf("a")) { }
        ✔ Because 0 is falsy

        // ✅ Correct
        if (str.indexOf("a") !== -1) { }
