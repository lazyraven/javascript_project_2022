1. What is includes method in JS ?

    Definition :  The includes() method in JavaScript is a powerful tool for checking the presence of a specific value within either an array or a string.
             It returns a boolean value (true or false) indicating whether the value is found.

  2. What is sort method in JS ?

                    1. String sorted , sort() : For basic alphabetical sorting where case doesn't matter or is consistent.
                      fruits.sort();

                2. Case-Insensitive Sort:,  localeCompare() : For robust, language-aware sorting, especially when dealing with mixed cases, special characters, or international languages.
                      fruits.sort((a, b) => a.localeCompare(b));

                3. Sorting Without Modifying the Original Array (ES2023): toSorted :  To create a new, sorted array without modifying the original (ES2023 and newer).
                       let newArray = fruits.toSorted();

                4. sort() + reverse() : A quick way to get descending order with basic alphabetical sorting.	
                       fruits.sort().reverse();
