• Explain this keyword in JS ? 

    Definition : In JavaScript, the this keyword is a special reference that points to the object that is currently executing the function. Its value depends on how the function is called, not where it is defined.
                        •  this depends on how a function is called.
                        • Arrow functions do not bind their own this.

                 1. Inside an Object Method : When this is used inside an object method, it refers to that object.
                
                2. In a Regular Function (Global Scope) : If you use this in a normal function (not inside an object), it depends:
                     • In strict mode → this is undefined.
                     • In non-strict mode → this refers to the global object (window in browsers, global in Node.js).

                 3. Inside a Constructor Function :  When using a constructor (with new), this refers to the newly created object.

                 4. In a Class : In classes, this works like in constructor functions — it refers to the instance of the class.

                 5. Arrow Functions : Arrow functions don’t have their own this. They inherit this from the surrounding scope.
                Arrow functions are often used when you want to preserve the this value from the outer scope.
