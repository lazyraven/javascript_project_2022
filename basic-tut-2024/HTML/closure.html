Imagine you have a function within another function and the inner function can access variables from the outer function even the outer function has finished executing. That's a closure!

Where are Closures Applied ?

Private Variables : encapsulating data within functions and preventing direct access from outside.

                    function counter() {
                        let count = 0;
                        return function () {
                        // return count++;
                        return ++count;
                        };
                    }
                    const increment = counter();
                    console.log(increment()); // o/p => 1
                    console.log(increment()); // o/p => 2

Iterators : Closure can be used to create iterators that maintain their state between calls, facilitating efficient and memory-friendly iteration over data structures.

                function createIterators(items) {
                    let index = 0;
                    return function () {
                      return items[index++];
                    };
                  }
                  
                  const fruitsIterator = createIterators(["apple", "banana", "orange"]);
                  console.log(fruitsIterator());
                  console.log(fruitsIterator());
