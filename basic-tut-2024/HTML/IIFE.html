IIFE - Immediately Invoked Functional Expressions

To view the output, you need to access the console tab of your browser's developer tools. - right click inspect mode

  • An IIFE is a function that is called immediately after it is definedd. 
    
                
    • 
                Use cases of IIFE :-
                
  • Avoid polluting the global namespace.
    • 
             
  • Execute async/await functions
    • 
             
  • Provides encapsulation, allowing you to create private scopes for variables and functions.
