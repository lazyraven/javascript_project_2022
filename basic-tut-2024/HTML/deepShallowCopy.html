Deep Shallow Copy

In JavaScript (and most programming languages), deep copy and shallow copy refer to how objects or arrays are duplicated in memory.

Shallow Copy => it'll change existing array when you'll modify copied array

                const original = { name: "Nisha", skills: ["JS", "React"] };
                const shallowCopy = { ...original };

                shallowCopy.name = "Updated";           // Only updates the copy
                shallowCopy.skills.push("Node.js");     // Also affects the original

                console.log(original.skills); // ['JS', 'React', 'Node.js']

Deep Copy => it'll not change into the existing array when you'll modify copied array.

                const original = { name: "Nisha", skills: ["JS", "React"] };
                const deepCopy = JSON.parse(JSON.stringify(original));

                deepCopy.skills.push("Node.js");

                console.log(original.skills); // ['JS', 'React']
                console.log(deepCopy.skills); // ['JS', 'React', 'Node.js']

In JavaScript, when you copy an object or an array, you can either create a shallow copy or a deep copy.

Shallow Copy

A shallow copy creates a new object or array, but it only copies the references to the nested objects or arrays. If you modify a nested object in the copied version, it will also affect the original object.

                const original = { a: 1, b: { c: 2 } };
                const shallowCopy = { ...original }; // Shallow copy using spread operator
                shallowCopy.b.c = 3; // Modifying nested object
                console.log(original.b.c); // Output: 3 (affected original object)

Deep Copy

A deep copy creates a new object or array and recursively copies all nested objects or arrays. Modifying a nested object in the copied version will not affect the original object.

                const original = { a: 1, b: { c: 2 } };
                const deepCopy = JSON.parse(JSON.stringify(original)); // Deep copy using JSON methods
                deepCopy.b.c = 3; // Modifying nested object
                console.log(original.b.c); // Output: 2 (original object remains unchanged)

In summary, a shallow copy only copies references to nested objects, while a deep copy creates a completely independent copy of the entire object structure.
