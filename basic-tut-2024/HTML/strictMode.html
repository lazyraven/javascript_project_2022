Strict Mode

To view the output, you need to access the console tab of your browser's developer tools. - right click inspect mode

The "use strict" directive enables Javascript strict mode. This was introduced in ECMAScript5. It enforces stricter parsing and error handling on the code at runtime. It also helps you write cleaner code and catch errors and bugs that might otherwise go unnotices. 

                    Eg:  // strict mode
                    "use strict";
                    
                    x = 56;
                    console.log(x);
