Currying

To view the output, you need to access the console tab of your browser's developer tools. - right click inspect mode

It involves breaking down a function that takes muliplt arguments into a series of functions that take one argument each.

Simple words:

This creates a chain of functions, where each function return another function until the final result is achieved.

            //currying
            // basic use case
                function sum(a) {
                return function (b) {
                    return function (c) {
                    return a + b + c;
                    };
                };
                }
                console.log(sum(3)(4)(5));

                // real life use case of event logger with time, type, message

                const logger = (time) => (type) => (message) =>
                `At ${time}, an event of type: ${type} occurred full details as: ${message}`;

                const eventsNow = logger("5am");
